API Documentation

This is the Application used by the Business Man to keep track of his transactions,payments,customers,type of products and the amount due by the customers.



Customer

HTML : "/customer/customer.html"

1. Creating a Customer

@POST : "/rest/customer"

HTML : "/customer/create.html"

This page will take the customer's information and store it to the database. The id of the customer which is called nid should be unique and it is a string. Once the nid is created you can't change the value.
Parameters consumed : { "nid": "", "name": "", "address": "", "pnumber": , "email": "", "date": "" }
The date should be in this formate : yyyy/mm/dd.

2. Updating a Customer

@POST : "/rest/customer/{nid}"

HTML : "/customer/update.html"

This page will give you the list of customers nid in a drop down Box then you can select one of the nid and update the data. you can't change the nid value.
Parameters consumed : { "name": "", "address": "", "pnumber": , "email": "", "date": "" }
The date should be in this formate : yyyy/mm/dd.

3. Display Customers

@GET : "/rest/customer"

HTML : "/customer/displayall.html"

This page will get all the customers in the table and desplay them.

@GET : "/rest/customer/get?nid='' "

HTML : "/customer/displaybyID.html"

This page will get the information of the customer whose nid is called.

@GET : "/rest/customer/get?name='' "

HTML : "/customer/displaybyName.html"

This will get the information of the customer whose name is called.

4. Delete customer

@DELETE : "/rest/customer"

HTML : "/customer/delete.html"

This page will take the nid of the customer and delete that customer

Transactions

HTML : "/tranasactions/tranasactions.html"

1. Creating Transactions

@POST : "/rest/transaction"

HTML : "/tranasactions/create.html"

This page will take the transaction information and send it to the database.
Parameters consumed : { "date": "2015-02-29", "nid": "mv", "type": "teja", "tid": 0, "bags": 5, "coolie": 5, "vat": 5, "rate": 130, "bagkg": [ 45,42.5,36.5,52.8,47 ],"railway": 6}

2. Updating a Transactions

@POST : "/rest/transaction/update/{tid}"

HTML : "/tranasactions/update.html"

This page will update the transaction information.
Parameters consumed : { "date": "2015-02-29", "nid": "mv", "type": "teja", "tid": 0, "bags": 5, "coolie": 5, "vat": 5, "rate": 130, "bagkg": [ 45,42.5,36.5,52.8,47 ],"railway": 6}

3. Display Transactions

HTML : "/tranasactions/displayall.html"

@GET : "/rest/transaction"

This will get all the transaction in the database.

@GET : "/rest/transaction/get?tid="

This will get the transaction based on ID.

@GET : "/rest/transaction/get?nid="

This will get the transaction based on nid.

@GET : "/rest/transaction/get?date="

This will get the transaction based on date.

@GET : "/rest/transaction/date/{date1}/{date2}"

This will get the transaction between the two date.

@GET : "/rest/transaction/date/{date1}/{date2}?nid="

This will get the transaction between the two date and the nid.

4. Delete Transactions

@DELETE : "/rest/transaction"

HTML : "/tranasactions/delete.html"

This page will ask for the ID of the transaction which user want to delete and deletes in the database.
